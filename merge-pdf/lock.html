Advanced: Owner Password & Permissions
The owner password unlocks full access — printing, editing and every permission below, no restrictions.
Why set an owner password? If you leave it blank, the document password doubles as the owner password — so anyone who opens the file automatically gets full permissions, and the toggles below won't really restrict anything. Set a separate owner password to make the permission limits below actually stick for people opening the file with the document password only.
